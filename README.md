# utools-Excel2Json

> 选择 excel 文件，转换输出 json 格式；可选择格式化 json 数据，一键复制

## 技术栈

vue3 + vite2 + utools

## 在线预览
[http://milo_admin.gitee.io/utools-excel2-json/](http://milo_admin.gitee.io/utools-excel2-json/)
